# Amazon Build Challenge Project

## Project Structure

```
amazon-build/
├── src/                 # Source code
├── assets/             # Game assets
│   ├── images/         # Sprites, backgrounds, UI elements
│   ├── sounds/         # Sound effects and music
│   └── fonts/          # Custom fonts
├── docs/               # Documentation
├── tests/              # Test files
└── README.md           # This file
```

## Getting Started

This project uses Python 3.12 and pygame 2.5.2.

### Prerequisites
- Python 3.12+
- pygame 2.5.2+

### Running the Game
```bash
cd src
python3 main.py
```
